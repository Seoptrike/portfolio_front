📋 프로젝트 소개

Seopotfolio는 누구나 회원가입하여 자신의 경력, 기술 스택, 프로젝트를 업로드할 수 있는 포트폴리오 플랫폼입니다.

개발자들이 자신의 역량을 효과적으로 어필할 수 있도록 다양한 에디터와 시각화 기능을 제공하며, 현대적인 서버리스 아키텍처로 구축되어 높은 가용성과 확장성을 보장합니다.

🎯 프로젝트 목표

이 프로젝트는 서버리스 배포 경험을 쌓기 위한 목적으로 시작되었습니다. 프론트엔드부터 백엔드, 데이터베이스, CI/CD까지 모든 과정을 서버리스 환경에서 구현하여 현대적인 웹 개발 및 배포 경험을 습득하는 것이 주요 목표였습니다.

🏗️서버리스 아키텍처

Frontend

React + Vite Vercel 배포

Backend

Spring Boot Docker Render 배포

Database

PostgreSQL Render 호스팅

DevOps

GitHub Actions Uptime Robot ImageKit

✨핵심 기능

  • 회원가입 및 프로필 관리 - 누구나 가입하여 포트폴리오 생성
  • 자기소개 에디터 - 리치 텍스트 에디터로 디자인된 자기소개 작성
  • 경력/기술서 작성 - Toast UI 마크다운 에디터 및 실시간 미리보기
  • 프로젝트 갤러리 - 프로젝트 업로드 및 관리
  • 기술 스택 자동 집계 - 프로젝트별 기술 스택 체크로 자동 별점 생성
  • 이미지 업로드 - ImageKit SDK 연동으로 최적화된 이미지 관리
  • 헬스체크 - Uptime Robot으로 10분마다 서버 상태 모니터링
  • CI/CD 파이프라인 - GitHub Actions로 자동화된 배포

📝에디터 기능

  • 자기소개 에디터 - 리치 텍스트 에디터로 폰트, 색상, 레이아웃 디자인
  • 마크다운 에디터 - Toast UI 에디터로 경력 및 기술서 작성
  • 실시간 미리보기 - 작성 중인 마크다운을 실시간으로 렌더링하여 확인
  • 코드 하이라이팅 - 기술 문서 작성 시 코드 블록 지원

⚙️자동화 기능

  • 기술 스택 자동 집계 - 프로젝트에 체크한 기술만큼 별점 자동 증가
  • 반응형 디자인 - 모바일부터 데스크톱까지 완벽 지원
  • 플로팅 네비게이션 - 사용자 친화적인 인터페이스
  • 이미지 최적화 - ImageKit을 통한 자동 이미지 최적화

🔮향후 개발 계획

  • GitHub API 연동 - 레포지토리를 직접 읽어서 자동으로 기술 스택 분석
  • OpenAI 연동 - AI가 자기소개서를 분석하여 메인 화면에 요약 제공
  • 추천 시스템 - 기술 스택 기반 개발자 추천 기능
  • 팀 매칭 - 프로젝트별 팀원 모집 및 매칭
  • 통계 대시보드 - 개인 포트폴리오 조회수 및 분석 제공
